发表于 2018-08-26 | 分类于 教程

SpringBoot

J2EE企业级开发的整套解决方案，可以让你更快速的去搭建一个可运行的项目。

B站教程

一、springboot 入门

原理 通过整合spring的全部技术栈来完成企业级开发

1、优缺点

优点

  • 快速创建与主流框架集成的能独立运行的Spring项目
  • 嵌入式的servlet容器，无需打成war包
  • starters自动依赖，版本控制
  • 大量的默认配置来简化开发，也可以修改默认配置
  • 无需XML，无代码生成的配置方式
  • 准生产环境的运行时应用监控
  • 与云计算天然集成

缺点

  • 入门容易精通难
  • springboot是对spring的再封装，只有对spring底层api有充分的了解才能用好springboot
2、微服务（一种架构风格）
  • 一组小型服务组成一个应用
  • 每一个小型服务都运行在自己的进程内
  • 服务之间可以通过HTTP沟通
  • 各个功能元素互相独立，可以互相组合
3、springboot项目
  • 通过idea的 spring initializr 建立
  • 相当于 一个maven工程 引入了springboot依赖

快速打成一个可执行的jar包

  • idea 右侧 maven project –> Lifecycle –> package

  • spring将所有的功能所需的依赖，整合成很多的starts
